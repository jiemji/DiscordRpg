Guide des Contrôles

Gestion de Scènes (Fichiers)

  • Sauvegarder : Télécharge un fichier .json contenant toute la scène sur votre ordinateur.
  • Charger : Ouvre l'explorateur pour choisir un fichier .json précédemment sauvegardé.
  • Nouvelle Carte : Charge une image et efface tous les pions existants.

Navigation

  • Bouger la carte : Clic Gauche maintenu + Glisser
  • Zoomer : Molette de la souris
  • Reset Vue : Réinitialise le zoom et la position.

Pions

  • Déplacer : Glisser-Déposer
  • Redimensionner : Tenir le Clic Gauche sur le pion + Molette
  • Supprimer : Tenir le Clic Gauche sur le pion + Touche Suppr
  • Changer Couleur : Double Clic sur l'image du pion
  • Nom du pion : Double Clic sur le nom pour le modifier
  • Copier/Coller : Ctrl+C / Ctrl+V sur le pion

Brouillard de Guerre

  • Activer : Bouton Œil
  • Dessiner : Mode Édition (Stylo) + Clic Gauche
  • Dissipation Auto : Le brouillard se dissipe automatiquement autour des pions.
