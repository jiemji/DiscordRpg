Guide des Contrôles

Navigation

  • Bouger la carte : Clic Gauche maintenu + Glisser
  • Zoomer : Molette de la souris
  • Reset Vue : Bouton

Pions

  • Déplacer : Glisser-Déposer
  • Redimensionner : Tenir le Clic Gauche sur le pion + Molette
  • Supprimer : Tenir le Clic Gauche sur le pion + Touche Suppr
  • Changer Couleur : Double Clic sur le pion

Brouillard de Guerre

  • Activer : Bouton
  • Dessiner : Mode Édition (Stylo) + Clic Gauche
  • Dissipation Auto : Le brouillard se dissipe automatiquement autour des pions lorsqu'on les bouge.
  • Taille des pions : Au chargement, les pions font 60px de diamètre. Prenez-le en compte lorsque vous créerez les maps. Une image en 1920 sur 1080 correspond environ à un rectangle de 33m sur 18m. Il est possible de modifier la taille des pions si besoin

Sauvegarde automatique activée dans ce navigateur.
