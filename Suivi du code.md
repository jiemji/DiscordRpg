#Suivi du code

## Phase 1
Le site doit permettre à l'utilisateur de charger une image. 
L'image est ensuite affichée pleine écran. le centre de l'image est affiché au centre de l'écran.
L'utilisateur peut zoomer ou dézoomer sur l'image. Le zoom se focalise sur le point du centre de l'écran.
L'image peut être tiré avec la souris de droite à gauche. L'utilisateur doit cliquer sur le bouton gauche et le maintenir enfoncé. 
- s'il tire le curseur vers la droite, le centre de l'image se déplace vers la droite. 
- S'il tire le curseur vers la gauche, le centre de l'image se déplace vers la gauche.
- S'il tire le curseur vers le haut, le centre de l'image se déplace vers le haut.
- S'il tire le curseur vers le bas, le centre de l'image se déplace vers le bas.
Le déplacement s'arrête quand l'utilisateur relache le bouton gauche de la souris
